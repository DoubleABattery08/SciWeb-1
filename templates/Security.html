Security Measures

At SciWeb, we take your security and privacy seriously. Here are some of the key security measures we've implemented to protect your data:

  • Password Hashing

    All passwords are securely hashed using SHA-256 before storage, ensuring that even in the unlikely event of a data breach, your password remains protected.

  • Grade Encryption

    All grade data is encrypted before storage using secure encryption keys, ensuring your academic information remains private. The keys are stored locally as a cookie, so not even we can access your grades.

  • API Key Protection

    All API keys are stored securely in separate configuration files and are never exposed to the client side or version control systems.

  • Google Cloud Protection

    Our database is hosted on Google Cloud, which is one of the most secure cloud platforms available. We use Google's robust security measures to protect your data.

  • HTTPS Protocol

    All communication between your browser and our servers is encrypted using HTTPS, protecting your data during transmission.

  • Access Control

    Robust authentication and authorization systems ensure users can only access their own data and permitted resources.

  • Regular Security Updates

    We maintain regular security updates and implement the latest security best practices to protect against emerging threats.
